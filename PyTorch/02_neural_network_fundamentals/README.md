## 🔬 物理视角

### 神经网络作为函数逼近器

从物理角度看，神经网络是一种**参数化的函数族**。训练过程就是在这个函数族中寻找最佳函数来拟合数据，类似于：

- 傅里叶级数展开：用正弦/余弦基函数逼近周期函数
- 球谐函数展开：用球谐函数逼近球面上的函数
- 神经网络：用激活函数作为"基函数"逼近任意函数

### 能量最小化视角

| 神经网络概念 | 物理类比 |
|-------------|---------|
| 损失函数 $L(\theta)$ | 系统能量/自由能 $E(\mathbf{r})$ |
| 模型参数 $\theta$ | 系统构型 $\mathbf{r}$ |
| 梯度 $\nabla_\theta L$ | 力的负方向 $-\mathbf{F}$ |
| 学习率 $\eta$ | 时间步长/阻尼系数 |
| 局部极小值 | 亚稳态 |
| 全局最小值 | 基态 |

### 统计力学视角

训练过程可以理解为一个采样过程：
- **确定性优化（SGD）** ≈ 零温极限下的能量最小化
- **随机梯度下降** ≈ 有限温度下的朗之万动力学
- **学习率退火** ≈ 模拟退火算法
