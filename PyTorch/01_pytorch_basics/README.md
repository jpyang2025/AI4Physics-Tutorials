# 第1章：PyTorch 基础与张量操作

## 📖 章节概述

欢迎来到 PyTorch 神经网络教程的第一章！本章将带你了解 PyTorch 的核心概念——**张量（Tensor）**和**自动微分（Autograd）**。

作为物理科研人员，你可能已经熟悉张量的数学概念（如应力张量、电磁场张量）。PyTorch 中的张量与此类似，是一种多维数组数据结构，但它还具备自动计算梯度的能力——这正是神经网络训练的基础。

## 🎯 学习目标

完成本章学习后，你将能够：

1. 创建和操作 PyTorch 张量
2. 理解张量与 NumPy 数组的关系和区别
3. 使用 GPU 加速张量计算
4. 理解并使用自动微分机制
5. 为后续的神经网络学习打下坚实基础

## 🔬 物理视角

在本章中，我们会用物理学的视角来理解这些概念：

| PyTorch 概念 | 物理类比 |
|-------------|---------|
| 张量 | 多维物理量（标量场、矢量场、张量场） |
| 张量运算 | 张量代数（收缩、外积等） |
| 自动微分 | 沿势能面计算力（负梯度） |
| 计算图 | 物理系统的因果关系链 |

## 📋 前置要求

- Python 3.8+
- 熟悉 NumPy 基本操作
- 了解微积分基础（求导、链式法则）

## 🚀 快速开始

```bash
# 确保已安装 PyTorch
pip install torch

# 运行示例代码
cd examples
python tensor_demo.py
python autograd_demo.py
```
